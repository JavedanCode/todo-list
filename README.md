# Todo List – The Odin Project

This project is a dynamic Todo List web application built as part of **The Odin Project** JavaScript curriculum.

It focuses on practicing modular JavaScript architecture, state management, DOM manipulation, and bundling with Webpack.

---

## 📚 About The Project

This application was built to simulate a real-world task management system with multiple projects and task organization.

The main goal was to strengthen understanding of:

- Modular JavaScript structure (separation of concerns)
- State management without frameworks
- Dynamic DOM rendering
- Event delegation
- LocalStorage persistence
- Webpack setup and configuration
- Scalable project architecture

---

## 🛠️ Built With

- JavaScript (ES6 Modules)
- Webpack
- HTML (template-based)
- CSS3 (custom design system)
- LocalStorage API
- Git & GitHub

---

## 🎯 Features

- Create, edit, and delete tasks
- Mark tasks as completed
- Organize tasks into multiple projects
- Switch between projects dynamically
- Filter tasks by:
  - All Tasks
  - Today
  - Completed
- Persistent data using LocalStorage
- Dynamic UI rendering system
- Modal system for tasks and projects
- Project deletion support
- Active state tracking for UI

---

## 🎯 What I Practiced

- Structuring a full application without frameworks
- Managing global state cleanly
- Avoiding tight coupling between logic and UI
- Using factory functions for data models
- Implementing a central render system
- Handling edge cases and data validation
- Writing scalable and maintainable code
- Debugging real-world UI/state issues
